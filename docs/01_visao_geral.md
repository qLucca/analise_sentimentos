# Visão Geral

Este projeto implementa tecnicamente o trabalho acadêmico sobre sentimentos de consumidores brasileiros em relação ao Nubank, com foco em atendimento ao cliente e segurança. A solução foi estruturada como pipeline de dados e NLP orientado a scripts Python, com SQL Server como camada analítica.

## Interpretador recomendado

- `Python 3.11`

Essa versão foi definida como base do projeto por compatibilidade com as bibliotecas de NLP, machine learning, dashboard e integração com SQL Server.
